# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текс курсивом, выдели его с обеих сторон "*". Например, *вот так*.

Чтобы выделить текст полужирным шрифтом, выдели его с обеих сторон двойными "**". Например, **Вот так**.
